DEPLOYMENT OPTIONS
================================================================================

OPTION 1: VERCEL (Recommended - Easiest)
  Benefits: Automatic SSL, fast, free tier
  Steps:
    1. npm install -g vercel
    2. vercel --prod
    3. Add environment variables in dashboard
  Time: 5 minutes

OPTION 2: NETLIFY
  Benefits: Simple, good performance, free tier
  Steps:
    1. npm install -g netlify-cli
    2. netlify deploy --prod --dir=dist
    3. Configure environment variables
  Time: 5 minutes

OPTION 3: DOCKER
  Benefits: Full control, scalable
  Steps:
    1. Create Dockerfile
    2. docker build -t ptoc-app .
    3. docker run -p 3000:3000 ptoc-app
  Time: 10 minutes

OPTION 4: TRADITIONAL HOSTING
  Benefits: Your infrastructure
  Steps:
    1. npm run build
    2. Upload dist/ to server
    3. Configure web server
    4. Set environment variables
  Time: Variable
